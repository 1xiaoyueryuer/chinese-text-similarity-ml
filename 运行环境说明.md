# 运行环境说明文档

## 一、项目概述

**项目名称**：基于 Hugging Face Embedding 模型的中文文本语义相似度分类研究

**项目作者**：潘煜（学号：246002918）

**课程名称**：机器学习

**项目简介**：本项目基于 Hugging Face 预训练 Embedding 模型提取文本特征，结合 SVM、Random Forest、XGBoost、Logistic Regression 四种传统机器学习分类器，对中文文本语义相似度进行二分类研究。实验在 LCQMC 数据集上验证，共测试 16 种模型-分类器组合。

## 五、运行说明

### 5.1 快速演示（无需下载数据，使用模拟数据）

```bash
python main.py --mode demo
```

- 不依赖网络，不下载预训练模型
- 运行时间：约 1-2 分钟
- 用于验证代码环境是否正确

### 5.2 完整实验（使用真实数据集）

```bash
python main.py --mode full
```

- 首次运行需要联网下载预训练 Embedding 模型（约 1-3 GB）
- 自动下载 LCQMC 数据集
- 运行时间：约 10-20 分钟（CPU，采样 5000 条数据）

### 5.3 重新提取特征（不使用缓存）

```bash
python main.py --mode full --no-cache
```

- 强制重新计算所有 Embedding 特征
- 运行时间较长

### 5.4 生成可视化图表

```bash
python visualization.py
```

- 根据实验结果生成对比图表
- 输出到 `results/` 目录

## 六、数据说明

### 6.1 数据集

- **LCQMC 数据集**：哈工大 LCQMC 中文语义相似度数据集
- 包含约 24 万条中文句子对，标注是否语义相似（0/1）
- 数据格式：`sentence1\tsentence2\tlabel`
- 数据自动下载到 `data/` 目录

### 6.2 预训练模型

实验使用以下 Hugging Face 预训练模型（首次运行自动下载）：

| 模型名称 | 模型 ID | 大小 |
|---------|--------|------|
| BGE-small-zh | `BAAI/bge-small-zh` | ~400 MB |
| BGE-base-zh | `BAAI/bge-base-zh` | ~1 GB |
| SBERT-multilingual | `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2` | ~400 MB |
| Text2Vec | `shibing624/text2vec-base-chinese` | ~400 MB |

模型下载后缓存到 `model_cache/` 目录，后续运行无需重复下载。

## 七、常见问题

### Q1：模型下载慢或失败？

国内网络访问 Hugging Face 较慢，建议：

```bash
# Windows PowerShell 设置镜像
$env:HF_ENDPOINT = "https://hf-mirror.com"
python main.py
```

或使用网络代理。

### Q2：内存不足？

修改 `config.py` 中的 `MAX_SAMPLES` 值（默认 5000），减小数据量：

```python
MAX_SAMPLES = 2000  # 减少到 2000 条
```

### Q3：显存不足？

修改 `config.py` 中的 `BATCH_SIZE`：

```python
BATCH_SIZE = 8  # 减小批次大小
```
